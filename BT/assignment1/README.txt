Set environment variable ANTLR_JAR to the file antlr-4.9.2-complete.jar in your computer
Change current directory to initial/src where there is file run.py
Type: py -3 run.py gen 
Then type: py -3 run.py test LexerSuite
Then type: py -3 run.py test ParserSuite
Then type: py -3 run.py test ASTGenSuite
Then type: py -3 run.py test CheckerSuite
Then type: py -3 run.py test CodeGenSuite
